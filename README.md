# Fluffy Game на Pixi.js

## Запуск

1. Откройте файл `index.html` в браузере (двойной клик или через команду npm start).
2. На экране появится анимированный кролик из вашего атласа.

## Структура
- `index.html` — стартовая страница, подключает Pixi.js и ваш скрипт.
- `game.js` — основной игровой код.
- `src/img/` — ваши атласы и изображения.

## Дальнейшие шаги
- Добавьте загрузку других атласов (мир, препятствия, эффекты).
- Реализуйте управление, столкновения, уровни и т.д.

**Pixi.js** — мощный движок для 2D-игр на JavaScript.
